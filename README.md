# IBM HR Analytics Dashboard

## Overview
This project focuses on workforce analytics and employee attrition analysis using Power BI. The dashboard helps HR teams identify factors contributing to employee turnover, evaluate workforce demographics, and support data-driven retention strategies.

## Business Problem
Employee attrition can negatively impact productivity, recruitment costs, and organizational performance. The objective of this dashboard is to uncover key patterns behind employee attrition and provide actionable insights for HR decision-making.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key Performance Indicators (KPIs)
- Total Employees
- Attrition Count
- Attrition Rate (%)
- Average Age
- Average Monthly Income
- Average Years at Company
- Job Satisfaction Score

## Dashboard Features
- Department-wise Attrition Analysis
- Gender Distribution Analysis
- Age Group Segmentation
- Job Role Performance Metrics
- Education Field Analysis
- Monthly Income Comparison
- Employee Satisfaction Trends

## Key Business Insights
- Identified departments with the highest employee attrition.
- Analyzed the relationship between salary and employee turnover.
- Evaluated job satisfaction impact on retention.
- Discovered age groups with higher resignation rates.
- Highlighted employee demographics influencing workforce stability.
